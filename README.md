# What-is-a-VLA-Vision-Language-Action-Model:
A VLA is an AI model that takes in visual input, uses language based reasoning, and outputs actions
How a VLA Works (Simple Breakdown):
Vision The model processes camera images (sometimes LiDAR too).
Language A language model interprets the scene, reasons about what’s happening, and predicts intent.
Action The system outputs a driving trajectory or control command.
Examples of VLA Models
•	Wayve Lingo 1 (industry)
•	DriveLM (research)
•	CoReVLA (research)
•	VLA based extensions of UniAD / VAD
